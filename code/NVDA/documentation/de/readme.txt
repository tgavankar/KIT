Allgemeines

Der NonVisual Desktop Access (NVDA) ist ein freier Screen Reader für Windows, der blinden und sehbehinderten Menschen gleichermaßen es ermöglicht, Ihren Computer ohne zusätzlichen Kostenaufwand nutzen zu können. NVDA erlaubt dem Benutzer herauszufinden, was auf dem Bildschirmpassiert, in dem es die Bildschirminformationen des Betriebssystems abfragt und eine Sprachausgabe und auch via Braillezeile zur Ausgabe von Informationen zur Verfügung stellt. 

Für weitere Informationen lesen Sie das beiliegende Benutzerhandbuch. Auf der NVDA-Webseite finden Sie ebenfalls mehr Informationen.
