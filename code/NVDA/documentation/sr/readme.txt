Невизуелни приступ десктопу (NonVisual Desktop Access) НВДА
URL: http://www.nvda-project.org/

Невузуелни приступ десктопу (НВДА) је бесплатан читач екрана отвореног кода за Мајкрософт Виндоуз (Microsoft Windows) оперативни систем. Овај софтвер може омогућити слијепим или слабовидим особама да користе компјутере, који користе Виндоуз као оперативни систем, исто као и особе које виде без додатног плаћања. НВДА омогућава корисницима да сазнају шта се догађа на екранау вршећи упите над оперативним системом и користећи синтетизатор говора за добијање излазних информација.

За више информација, молимо вас погледајте корисничко упутство. Додатне информације и ресурси се могу наћи на веб сајту НВДА.
