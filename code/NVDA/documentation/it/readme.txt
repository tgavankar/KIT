NonVisual Desktop Access (NVDA)
URL: http://www.nvda-project.org/

NonVisual Desktop Access (NVDA) è uno screen reader open-source e completamente gratuito per il sistema operativo Windows che consente ai disabili visivi di utilizzare il computer senza sostenere ulteriori costi oltre a quelli del pc e del sistema operativo. Nvda consente all'utente di capire cosa succede sullo schermo che viene letto attraverso la sintesi vocale. 

Per ulteriori informazioni consultate la guida dell'utente o visitate il sito di Nvda.
