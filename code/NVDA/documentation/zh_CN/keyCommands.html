NVDA 2012.1 快捷键快速参考

开始使用 NVDA

基本快捷键

名称 快捷键 描述
停止朗读 Ctrl 立即停止朗读。
暂停/继续朗读 Shift 立即暂停朗读，再按一次会从原来暂停的地方继续朗读(若使用的语音合成器有支持的话)。
NVDA 菜单 NVDA+n 打开 NVDA 菜单以访问选项、工具及辅助描述等。
快捷键描述模式 NVDA+1 在这个模式下，按下任何按键组合，即会朗读出相关的 NVDA 的按键名称与功能描述。
关闭 NVDA NVDA+q 关闭 NVDA。
读出日期或时间 NVDA+F12 按一次读出当前时间，连按两次读出当前日期。
忽略下一个NVDA快捷键 NVDA+F2 告知 NVDA 忽略运行下一个 NVDA 的组合按键，即使此指令通常是 NVDA 的一个快捷键。

NVDA 导航

系统焦点导航

名称 快捷键 描述
读出当前焦点 NVDA+tab 读出当前焦点所在的对象或控件，连按两次将拼读其信息。
读出窗口标题 NVDA+t 读出当前作用中的窗口标题，若连按两下 T 将标题拼读出来，连按三下T 则将标题复制到剪贴板。
读出作用中的窗口内容 NVDA+b 读出当前作用中窗口所有的控件(尤其在对话框中很有用)。

系统编辑光标导航

名称 快捷键 描述
向下读出全部 NVDA+向下键 从当前光标所在的位置往下读出直到末尾，光标会跟随移动。
读出游标所在行的内容 NVDA+向上键 读出游标所在行的内容，连按两次会拼读出这一行内容。
读出选择的内容 Shift+NVDA+向上键 读出当前已选择的文本。
读出字符的格式信息 NVDA+f 读出当前光标所在位置字符的格式信息。

当编辑光标在一个表格内时，也可使用下列的快捷键：

名称 快捷键 描述
移到左一列 Ctrl+Alt+向左键 移动光标到左一列 (仍在同一行中)。
移到右一列 Ctrl+Alt+向右键 移动光标到右一列 (仍在同一行中)。
移到上一行 Ctrl+Alt+向上键 移动光标到上一行 (仍在同一列中)。
移到下一行 Ctrl+Alt+向下键 移动光标到下一行 (仍在同一列中)。

对象导航

名称 台式电脑快捷键 笔记本电脑快捷键 描述
读出当前对象 NVDA+数字键盘5 Ctrl+NVDA+i 读出当前的对象，连按两下拼读出此对象信息，连按三下则将对象的内容复制到剪贴板。
浏览至父对象 NVDA+数字键盘8 Shift+NVDA+i 浏览至当前对象的父对象（也就是包含此对象的容器）。
浏览至上一个对象 NVDA+数字键盘4 Ctrl+NVDA+j 直接刘览至当前所在对象的上一个对象。
浏览至下一个对象 NVDA+数字键盘6 Ctrl+NVDA+l 直接浏览至当前所在对象的下一个对象。
浏览至子对象 NVDA+数字键盘2 Shift+NVDA+逗点 浏览至包含于当前对象内的第一个子对象。
将浏览光标移至焦点对象 NVDA+数字键盘减号 NVDA+backspace 将浏览光标移至当前系统焦点所在的对象。
启动当前浏览的对象 NVDA+数字键盘Enter NVDA+enter 启动当前浏览的对象 (与在系统焦点下按下鼠标或空格键类似)。
将系统焦点移到当前浏览的对象上 Shift+NVDA+数字键盘减号 Shift+NVDA+backspace 假如可以的话，将系统焦点移到当前浏览的对象上。
读出浏览对象的尺寸大小 NVDA+数字键盘 Delete NVDA+Delete 读出当前浏览的对象在屏幕上所占尺寸大小的百分比(包含与屏幕左侧与上缘的距离以及它的宽度与高度)。

在当前的对象下浏览文本

名称 台式电脑快捷键 笔记本电脑快捷键 描述
移到第一行 Shift+数字键盘7 NVDA+7 将浏览光标移动到文档中的第一行。
移到上一行 数字键盘7 NVDA+u 移动浏览光标到所在位置的上一行并读出该行文本。
读出浏览光标所在行 数字键盘8 NVDA+i 读出浏览光标所在的一行文本，连按两次拼读出整行。
移到下一行 数字键盘9 NVDA+o 移动浏览光标到所在行的下一行并读出该行文本。
移到最后一行 Shift+数字键盘9 NVDA+9 移动浏览光标到文档中的最后一行。
移到上一个字 数字键盘4 NVDA+j 移动浏览光标到文档中的前一个文本并读出。
读出浏览光标所在的字 数字键盘5 NVDA+k 读出当前浏览光标所在的一个字，连按两下拼读出这个字（附注：此处是指英文的一个单字）。
移到下一个字 数字键盘6 NVDA+l 移动浏览光标到文档中的下一个字，并将其读出。
移到行首 Shift+数字键盘1 Shift+NVDA+u 移动浏览光标到当前所在行的行首。
移到前一个字符 数字键盘1 NVDA+m 移动浏览光标到当前所在位置的前一个字符并读出。
读出浏览光标所在的字符 数字键盘2 NVDA+逗点 读出浏览光标所在的字符，按两下则读出此字符的内码。
移到下一个字符 数字键盘3 NVDA+句点 移动浏览光标到当前所在位置的下一个字符并读出。
移到行尾 Shift+数字键盘3 Shift+NVDA+o 移动浏览光标到当前所在行的行尾。
向下全部朗读 数字键盘加号 Shift+NVDA+向下键 从浏览光标当前所在位置往下读出，浏览光标跟着移动。
移动编辑光标到浏览光标 Ctrl+数字键盘减号 Ctrl+NVDA+backspace 移动编辑光标到浏览光标的位置。
从浏览光标开始复制 NVDA+F9 NVDA+F9 将浏览光标所在字符设置为复制区段的起点，实际复制的动作将等到末尾位置的设置后才会进行。
复制至浏览光标 NVDA+F10 NVDA+F10 将先前一个指令所设置的起点到当前光标所在位置间的区段复制到剪贴板中。

平面浏览

名称 台式电脑快捷键 笔记本电脑快捷键 描述
切换到平面浏览 NVDA+数字键盘7 NVDA+PageUp 切换到平面浏览模式，将浏览光标停在当前浏览对象的位置上，如果当前是在浏览文档的情况下，则用户可利用浏览光标来浏览整个文档窗口，包括标题列及工具栏的项目。
从平面浏览切换到对象上 NVDA+数字键盘1 NVDA+PageDown 从平面浏览模式切换到对象上。

鼠标浏览

名称 台式电脑快捷键 笔记本电脑快捷键 描述
单击鼠标左键 数字键盘除号 NVDA+向左键 单击鼠标左键一次。连续快按二下为双击鼠标左键。
锁定鼠标左键 Shift+数字键盘除号 Shift+NVDA+向左键 锁定鼠标左键，再按一次则解锁。此功能可模拟按住鼠标左键，运行拖曳的动作。
单击鼠标右键 数字键盘乘号 NVDA+向右键 单击鼠标右键一次。
锁定鼠标右键 Shift+数字键盘乘号 Shift+NVDA+向右键 锁定鼠标右键，再按一次则解锁。此功能可模拟按住鼠标右键，运行拖曳的动作。
移动鼠标到当前浏览对象 NVDA+数字键盘除号 Shift+NVDA+F9 移动鼠标到当前浏览对象的浏览光标位置。
浏览移到鼠标所在的对象 NVDA+数字键盘乘号 Shift+NVDA+F10 将浏览对象设置到当前鼠标所指的对象。

虚拟缓冲

名称 快捷键 描述
切换虚拟缓冲模式 NVDA+空格 切换焦点模式与浏览模式。
离开焦点模式 Esc 由先前的焦点模式切换回浏览模式。
刷新虚拟缓冲 NVDA+F5 重新加载当前虚拟缓冲的内容 (若当前页面的内容象似有遗漏时可运行此功能)。
寻找 Ctrl+NVDA+f 打开一个对话框，以便输入字符串，在当前的文档中查找。
寻找下一个 NVDA+F3 在当前文档中向下寻找前次查找的文本。
寻找上一个 Shift+NVDA+F3 在当前文档中向上寻找前次查找的文本。

单键浏览

下列的按键，若仅按该单一键，则会跳至其相应的下一个元素位置，如果是同时按 Shift+该单一按键，则会跳至前一个元素位置。

  • h: 标题
  • l: 列表
  • i: 列表项目
  • t: 表格
  • k: 链接
  • n: 非链接文本
  • f: 表单
  • u: 为访问的链接
  • v: 以访问的链接
  • e: 编辑区
  • b: 按钮
  • x: 复选框
  • c: 组合框
  • r: 单选按钮
  • q: 引述区
  • s: 分隔符
  • m: 页框
  • g: 图片
  • d: 书签
  • o: 嵌入式对象
  • 1 到 6: 第一层级到第六层级标题

元素列表

名称 快捷键 描述
虚拟缓冲元素列表 NVDA+F7 显示元素列表的对话框，列出当前虚拟缓冲中所有的链接项目、标题及书签等。

嵌入式对象

名称 快捷键 描述
离开到包含对象的虚拟缓冲 Ctrl+NVDA+空格 焦点离开当前的嵌入式对象回到包含该对象的虚拟缓冲。

NVDA 设置

偏好设置

语音设置 (Ctrl+NVDA+v)

名称 台式电脑快捷键 笔记本电脑快捷键 描述
读出所有标点符号 NVDA+p NVDA+p 当这项功能被激活时，所有的标点符号均会被读出，此功能对文本编校甚有用。

快速语音设置

名称 台式电脑快捷键 笔记本电脑快捷键 描述
切换到下一个语音控件 Ctrl+NVDA+向右键 Ctrl+NVDA+向右键 切换到下一个语音控件，包括语音、速度、音调及音量。若已到最后一个，再按则会回到第一个设置控件。
切换到上一个语音控件 Ctrl+NVDA+向左键 Ctrl+NVDA+向左键 切换到上一个语音控件，若已到第一个则会回到最后一个设置。
增加当前语音控件的设置 Ctrl+NVDA+向上键 Ctrl+NVDA+向上键 增加当前使用的语音控件的设置值，如选择前一个语音、增快速度、提高音调及调大音量。
减少当前语音控件的设置 Ctrl+NVDA+向下键 Ctrl+NVDA+向下键 减少当前使用的语音控件的设置值。

点字设置

名称 台式电脑快捷键 笔记本电脑快捷键 描述
点字光标跟随 Ctrl+NVDA+t Ctrl+NVDA+t 这个功能可切换显示在点字显示器上的消息，为一个切换指令开关，两种选择分别是： 一、显示编辑光标或焦点所在的内容，也就是移动方向键或 Tab 所指的位置；二、显示浏览光标所指的位置，也就是利用对象导航所送出的消息。

键盘设置 (Ctrl+NVDA+k)

名称 台式电脑快捷键 笔记本电脑快捷键 描述
读出输入字符 NVDA+2 NVDA+2 当激活时， NVDA 将读出所有在键盘上输入的字符。
读出输入字词 NVDA+3 NVDA+3 当激活时， NVDA 将读出所有在键盘上输入的字词，此处所指的字词，在英文是一个单字，在中文是以标点符号或空格隔开的一段字符串。
读出指令键 NVDA+4 NVDA+4 当激活时， NVDA 将读出指令键名称， 这包含组合键如 Ctrl 加上其它字母。

鼠标设置 (Ctrl+NVDA+m)

名称 台式电脑快捷键 笔记本电脑快捷键 描述
启动读出鼠标所指文本 NVDA+m Shift+NVDA+m 激活后，当移动屏幕上的鼠标时，NVDA 将读出鼠标所指的文本。这将可使你在屏幕上藉由移动鼠标来寻找对象，而不是藉由对象浏览的方式。

浏览光标设置

名称 台式电脑快捷键 笔记本电脑快捷键 描述
跟随系统焦点移动 NVDA+7 Ctrl+NVDA+7 激活后，浏览光标会跟随系统焦点移动。
跟随编辑光标移动 NVDA+6 NVDA+6 激活后，浏览光标会跟随编辑光标移动。

对象显示设置 (NVDA+control+o)

名称 台式电脑快捷键 笔记本电脑快捷键 描述
读出进度蓝 NVDA+u Ctrl+NVDA+F2 这个选项控制 NVDA 如何朗读出进度蓝的最新情况。

虚拟缓冲设置 (Ctrl+NVDA+b)

名称 台式电脑快捷键 笔记本电脑快捷键 描述
使用屏幕实际配置(当有支持时) NVDA+v NVDA+v 此复选框决定是否让虚拟缓冲区显示的内容与屏幕上的编排方式相同或是将链接项单独列在一行。若选择此项，则文本排列方式与屏幕上相同。

保存与撤消设置

名称 台式电脑快捷键 笔记本电脑快捷键 描述
保存设置 Ctrl+NVDA+c Ctrl+NVDA+c 保存当前的设置，当离开 NVDA 时，设置的数据不会丢失。
撤消设置 Ctrl+NVDA+r Ctrl+NVDA+r 将 NVDA 回滚到最后一次保存的设置。
