เอกสาร NVDA 

โปรแกรมอ่านหน้าจอสำหรับระบบปฏิบัติการ Windows  ซึ่งเป็นโอเพ่นซอร์สโปรแกรมที่ให้ใช้ฟรี

เขียนโดย Michael Curran และได้ความช่วยเหลือจาก James Teh และนักพัฒนาคนอื่นๆ

NonVisual Desktop Access (NVDA) คือโปรแกรมอ่านหน้าจอแบบโอเพ่นซอร์สสำหรับระบบปฏิบัติการ Windows ซึ่งช่วยให้ผู้ที่มีปัญหาทางด้านสายตาสามารถใช้คอมพิวเตอร์ได้โดยไม่ต้องเสียค่าใช้จ่ายใดๆทั้งสิ้น โปรแกรมนี้เริ่มพัฒนาในเดือนเมษายน ปีค.ศ. 2006 และยังคงพัฒนาให้มีประสิทธิภาพมากขึ้นอย่างต่อเนื่อง ทำให้มีผู้นำมาใช้งานกันอย่างแพร่หลาย ซึ่ง NVDA อาจจะไม่เสถียรหรือมีฟีเจอร์พิเศษเหมือนกับโปรแกรมอ่านหน้าจอที่ทำออกมาในรูปแบบธุรกิจตัวอื่นๆ แต่ตั้งแต่เดือนธันวาคม ค.ศ. 2006 ทางผู้พัฒนา NVDA ได้ทุ่มกำลังและลงมือแก้ไข ปรับปรุงแบบ full-time เพื่อที่จะให้มีคนนำมาใช้งานมากขึ้นและสุดท้ายผู้คนเหล่านั้นก็จะเลิกใช้โปรแกรมอ่านหน้าจอที่ทำออกมาในรูปแบบธุรกิจเหล่านั้นในที่สุด

NVDA สามารถทำงานร่วมกับส่วนต่างๆของระบบปฏิบัติการได้เป็นอย่างดี เช่น

ท่องเว็บเพจ (โดยใช้ Internet Explorer และ Mozilla Firefox)
อ่านและเขียนเอกสารด้วยโปรแกรมเช่น WordPad หรือ Microsoft Word
รับหรือส่งอีเมลล์ด้วย Outlook Express 
สามารถใช้ command-line ใน Dos ของ windows
สามารถทำงานร่วมกับ spreadsheets ใน Microsoft Excel
สามารถจัดการทั่วๆไปผ่าน My Computer, Windows Explorer, Control Panel และ งานทั่วๆไปของ Windows 

NVDA ไม่ได้หยุดด้วยเทรนด์และความต้องการในตลาดเลย ถึงแม้ว่าโปรแกรมอ่านหน้าจอที่ถูกผลิตออกมาในรูปแบบธุรกิจนั้นจะมีฟีเจอร์ที่ดีกว่า แต่บางครั้งก็ไม่เป็นไปตามความต้องการของตลาด NVDA ไม่ได้เป็นแค่โปรแกรมอ่านหน้าจออย่างเดียวเสมอไป แต่มันสามารถใช้เป็นเครื่องมือสำหรับทดลองบางอย่างได้ ซึ่งมันสามารถทำการทดสอบได้อย่างง่ายดาย รวดเร็วโดยที่โปรแกรมอ่านหน้าจอตัวอื่นๆไม่สามารถทำได้ เช่น ความสามารถในการใช้เสียงเตือนเพื่อสื่อสารกับผู้ใช้ในขณะที่ progress bar กำลังทำงาน เมื่อสัญญาณเสียงสูงมากขึ้น แสดงว่า ขั้นตอนในการทำงานใกล้เสร็จสิ้นแล้ว

คุณสามารถช่วยพัฒนา NVDA โดยแจ้งปัญหาที่พบ ข้อแนะนำเมื่อพบข้อผิดพลาดของโปรแกรมและบทความต่างๆสามารถดูได้ที่ www.nvda-project.org/nvda/wiki. หรือ คุณสามารถค้นหาข้อมูลทั่วไปเกี่ยวกับ NVDA ได้ที่เว็บไซต์ www.nvda-project.org/nvda และที่เว็บไซต์นี้คุณสามารถทราบรายละเอียดต่างๆโดยสมัครเป็นสมาชิก mailling list ของ NVDA โดย mailing list นั้นมี 2 แบบ คือ แบบที่ 1 จะสนทนากันในเรื่องทั่วๆไป ส่วนแบบที่ 2 จะสนทนากันในหมู่ผู้พัฒนาหรือผู้ที่นำโปรแกรมไปทดสอบ
